## 1. Network Connection Setup
Connect devices to the switch as follows:
- **Port 1** → Connect to the White LAN (Internet access required).
- **Port 2** → Connect to the NVR (e.g., UCK G4 Plus).
- **Ports 3–8** → Connect cameras and the setup PC.

## 2. Verify Network Connectivity
On the setup PC:

1. Disable Wi-Fi to ensure testing is performed through the Ethernet connection only.
2. Open **Command Prompt (CMD)**.
3. Run the following command to verify network settings:
    
    ```
    ipconfig
    ```
4. Verify connectivity to the NVR using:
    
    ```
    ping XX.XX.XX.XX
    ```
    
    Replace `XX.XX.XX.XX` with the NVR's IP address.
